# UXR Skills

UXRチームで共有する Claude Code スキル集。

## 使い方

1. このリポジトリをクローン
2. 使いたいスキルを `.claude/skills/` にコピーまたはシンボリックリンクで配置
3. Claude Code でスラッシュコマンドとして呼び出し

## スキル一覧

| スキル名 | 概要 | コマンド |
|---------|------|---------|
| uxr-hot-topics | UXR Insight Card を生成し、ドメイン別にフィルタ可能な index.html で一覧化 | `/uxr-hot-topics` |
